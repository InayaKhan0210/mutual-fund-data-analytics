# Mutual Fund Data Analytics

A data analytics project focused on mutual fund industry data, including NAV history, fund performance, SIP inflows, AUM trends, portfolio holdings, and benchmark analysis.

## Project Structure

* `data/raw` – Raw datasets and API data
* `data/processed` – Cleaned datasets
* `notebooks` – Jupyter notebooks
* `sql` – SQL queries
* `dashboard` – Dashboard files
* `reports` – Project reports and summaries
